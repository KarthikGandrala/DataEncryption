Information:

The basic rotation encryption method is a simple method to
conceal data by shifting each alphanumeric character by a
specified numerical value between 0 and 26. For example, the world
'hello' shifted by '1' would become 'ifmmp'. This can be used to transfer
non essential data via a more secure method.











Information:

The Poly Alphabetic Encryption method is much more secure method
of transferring data. This encryption method utilies the vigenere square
as well as takes a string to Encrypt the data instead of a numerical key
and has multiple advantages. For one, information with the same letters will
not be the same when encrypted. For example, when 'hello world' is encryptyed
with the string 'qwerty', it becomes 'xapch uenpu'. Even though it has 2 'o',
they become 'h' and 'e' respectively.











Information:

Using the XOR algorithm to encrypt data outputs messages that are much more garbled and
difficult to predict. It hides more information than a simple rotation cipher.
XOR also uses binary, but the algorithm does not add like in the rotation cipher. It compares each
corresponding bit, and outputs 0 if they are the same, 1 if they are different
The XOR cipher takes a key of any length for added security and is nearly impossible to break
